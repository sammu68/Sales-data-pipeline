This project is a simple yet practical Data Engineering pipeline built using Python that demonstrates the core concepts of ETL (Extract, Transform, Load). The pipeline reads raw sales data from a CSV file, processes and cleans the data by handling missing values and correcting data types, and then performs transformations such as calculating revenue and organizing the dataset for analysis. After processing, the cleaned data is stored in a structured format, and a summary report is generated to provide insights such as daily sales and total revenue. This project showcases essential data engineering skills including data ingestion, data cleaning, transformation, aggregation, and output generation using libraries like Pandas. It is designed to be beginner-friendly while also serving as a strong foundation for more advanced data pipeline projects involving databases, cloud platforms, and workflow automation tools.
